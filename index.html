Automated GCC Differences Reports

January 2026

• Automated solution for cross-workbook transaction validation, mismatch detection, and exception reporting

Built an Excel VBA automation tool that compares two banking transaction workbooks using normalized transaction reference keys to identify reconciliation exceptions with zero manual processing. The tool auto-detects headers and relevant columns across sheets, handles inconsistent formatting (spaces, commas, scientific notation, leading zeros), and generates structured report sheets for key scenarios including missing approved transactions, decline cases existing in both files while excluding reversal duplicates, and reversal mismatch detection. Designed as a single macro-enabled workbook with a button-driven workflow for selecting input files and producing clean audit-ready outputs.

Cloudflare IP Range Monitoring & Alerting System

December 2025

• Serverless solution for automated Cloudflare IPv4 and IPv6 IP range monitoring and alerting

Fully serverless AWS automation that monitors Cloudflare IP range updates using Amazon EventBridge and AWS Lambda. The solution fetches Cloudflare IP ranges via the public API, compares them with the previous version stored in Amazon S3, and sends automated email alerts through Amazon SES when IP changes are detected.

CloudWatch Widget Rotation and Alerting Automation System

November 2025

• Serverless solution for automated CloudWatch dashboard rotation and real-time alerting

Fully serverless monitoring solution using EventBridge, AWS Lambda, S3, CloudWatch Dashboards, and SES to dynamically rotate dashboard widgets and centralize alert visibility. Widget configurations are grouped and stored in S3, then rotated automatically on a dynamic dashboard. The system continuously scans CloudWatch alarms, updates a dedicated alert dashboard, and sends automated email notifications for critical alarms, improving operational visibility and incident response efficiency.

EC2 Auto Tagging

May 2025

• Serverless solution for automated EC2 tagging and compliance alerts

Fully serverless EC2 auto-tagging solution using CloudTrail, EventBridge, and three Lambda functions to apply metadata such as creator email and department. Untagged instance data is stored in DynamoDB, and Amazon SES sends automated alerts for missing tags.
